## ②課題内容（どんな作品か）
このアプリは、認知行動療法（CBT）の 「コラム法」 をベースにした、
やさしく気持ちを整理するための“心のメモ帳” です。

・今日あった「状況」
・そのときの「感情」
・ふっと浮かんだ「自動思考」
・気づいた「認知の歪み」
・もう少しやさしい「客観視点」
・その結果どう気持ちが変わったか——

これらを1枚の付箋カードのように記録することで、
自分の心の動きが見える化され、少しラクになること を目指しています。

## ⑤工夫した点・こだわった点
・ノート風デザイン
入力欄・一覧欄ともに「罫線のあるノートの質感」を再現し、落ち着いて書き込みやすい雰囲気にした。

・手書き風フォント
「Yomogi」フォントを採用し、きつさのない、やさしく寄り添う空気感を演出。

・付箋カード風レイアウト
記録は 1 件ずつ“付箋”のようなカードで表示。
過度に情報量を詰めず、心理的な負担が少ない見やすさに調整。

・感情ジャンルごとの色分け
ネガティブ：青
ニュートラル：緑
ポジティブ：黄色
カード背景そのものをジャンル色で表示し、直感的に「気持ちの傾向」が掴めるようにした。

・必要なときだけ一覧が表示される UI
記録が 0 件のときは一覧を非表示にし、
初回の入力後にふわっと一覧エリアが現れる仕組み。
画面の圧迫感をなくし、やさしい体験を意識した。

## ⑥難しかった点・次回トライしたいこと（又は機能）
過去の記録の閲覧機能
・カレンダー表示や、特定の日付をクリックしてその日の記録だけを見られる UI も検討したい。

客観視点（リフレーム）の AI 自動生成
・入力された「状況」「自動思考」「認知の歪み」を元に、AI が客観的な見方や優しい言い換えを自動提案する機能を追加したい。
・人がしんどいときでも、自分ひとりで考え込まずに、アプリ側から「こんな見方もあるよ」とそっと寄り添ってくれるような使い方を目指したい。
